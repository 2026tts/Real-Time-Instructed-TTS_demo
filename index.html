Abstract

Instruction-guided text-to-speech provides a flexible way to specify desired speaking style with natural language. However, existing open-source instruction datasets often rely on limited template expressions, and many prior models adopt autoregressive decoding with relatively high inference cost. In this work, we build Instruction-Voice, a bilingual multi-form instruction dataset with generation and rewriting pipelines, and we introduce Swift-Instruct-Voice (SI-Voice), an efficient non-autoregressive TTS model for low-latency instruction-guided speech synthesis. SI-Voice supports both strong instruction-following generation and competitive zero-shot bilingual synthesis.

Instruction-Voice Dataset
Built with two pipelines across Chinese and English, with six instruction styles for each utterance: Natural, Expressive, and Label in both languages.
SI-Voice
  • A bilingual speech synthesis model that supports both instruction-guided synthesis and zero-shot synthesis within one framework.
  • Real-time non-autoregressive synthesis with a compact architecture for efficient low-latency generation.

Instruction-Voice Dataset

We build Instruction-Voice, a bilingual multi-form instruction dataset for instruction-guided TTS, covering around 1.3k hours of speech with six stylistic instruction variants for each utterance in English and Chinese.
